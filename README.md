# WarhornRuby

This gem will function as a Ruby wrapper for the Warhorn API v2 ( https://github.com/warhorn/api/tree/v2 ).

## Installation

Add this line to your application's Gemfile:

    gem 'warhorn_ruby'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install warhorn_ruby
